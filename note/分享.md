
## f能取到a,b吗？原理是什么？
var F = function(){};
Object.prototype.a = function(){};
Function.prototype.b = function(){};
var f = new F();

































f.a === Object.prototype.a   //=> true
f.b === Function.prototype.b  //=> false
f 的原型链：

f -----> F.prototype -----> Object.prototype -----> null
F 的原型链：

F -----> Function.prototype -----> Object.prototype -----> null
